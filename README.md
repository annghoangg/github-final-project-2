# Paradise Nursery Shopping Application

A dynamic React-based e-commerce application for an online plant shop. 
Users can browse various categories of houseplants, add them to a shopping cart, and manage their selections.

## Features
- Dynamic landing page with "Get Started" functionality.
- Product listing with 3+ categories and 18+ unique plants.
- Redux-based shopping cart management.
- Responsive design for a seamless shopping experience.
